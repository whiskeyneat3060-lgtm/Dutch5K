# Dutch 5K

Mondrian-styled Dutch vocabulary trainer. Single-file app (`index.html`), deployed via Cloudflare Pages (auto-deploys from `main`).

- 5,000 highest-frequency Dutch words baked in (wordfreq), most-common-first
- Flashcards with labeled grammar forms, synonyms/antonyms, example sentences
- Cards auto-enrich via the Anthropic API (key entered on the Progress tab, stored only in the browser)
- Progress saved locally; Export/Import for backups
